# Kamal Evergreen School System Website

A modern, highly professional, and mobile-responsive website designed for 'Kamal Evergreen School System'. This project blends traditional values with future-ready skills (AI/Coding).

## Key Features
- **Mobile-First Design**: Optimized for smartphones with a responsive hamburger menu and vertical stacking for mobile users.
- **Premium Aesthetics**: Using a color palette of Deep Emerald Green, Professional Gold, and Clean White.
- **Interactive Hero Section**: Featured call-to-action buttons for immediate engagement.
- **AI-Skilled Faculty**: Highlighted transition towards modern educational standards.
- **Simple Admission Form**: A streamlined 1-page form for higher conversion on mobile devices.
- **Floating WhatsApp**: Instant communication link for parents.

## Technologies Used
- HTML5 & Tailwind CSS (CDN-based for fast delivery)
- JavaScript (Vanilla) for UI logic
- Lucide Icons for modern, lightweight iconography
- Google Fonts (Inter & Playfair Display)

## How to Run
Simply open `index.html` in any modern web browser.

## File Structure
- `index.html`: Main website file.
- `images/`: Directory containing generated high-quality assets.
